# Vim Basic
## Syntax
1. define variable
* let var_name = value
	* b:name          缓冲区的局部变量
	* w:name          窗口的局部变量
	* g:name          全局变量 (也用于函数中)
	* v:name          Vim 预定义的变量
* unlet var_name
2. key notion
* <BS>           Backspace
* <Tab>          Tab
* <CR>           Enter
* <Enter>        Enter
* <Return>       Enter
* <Esc>          Escape
* <Space>        Space
* <Up>           Up arrow
* <Down>         Down arrow
* <Left>         Left arrow
* <Right>        Right arrow
* <F1> - <F12>   Function keys 1 to 12
* #1, #2..#9,#0  Function keys F1 to F9, F10
* <Insert>       Insert
* <Del>          Delete
* <Home>         Home
* <End>          End
* <PageUp>       Page-Up
* <PageDown>     Page-Down
* <bar>          the '|' character, which otherwise needs to be escaped '\|'   
Commonly used:
* <C-key> 	Ctrl+key
* <S-key>	Shift+key
* <A-key> or <M-key> 	Alt+key
3. use help
* :help something<tab> --can help list out related topics
* :vert help something --to open window vertically
